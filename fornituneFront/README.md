# Iniciando a aplicação

## Funções

Verifique que você está na pasta correta: 

### `cd fornituneFront`

Para instalar os pacotes necessários, digite no console:

### `npm install`

Com o processo de instalação dos pacotes concluido, digite:

### `npm start`

Após este comando, a aplicação estará pronta para funcionar. Habilitando automáticamente a porta:
http://localhost:3000

Nesta aplicação é possível observar a tela de apresentação do sistema, com acesso para a tela de login.
Podemos perceber que alguns botões são clicáveis, e nos redirecionam para algumas rotas pela aplicação.
Porém, estas rotas não estão disponíveis no momento.

Segue o link de Deploy da aplicação na nuvem: https://front-end-fornitune.vercel.app/
